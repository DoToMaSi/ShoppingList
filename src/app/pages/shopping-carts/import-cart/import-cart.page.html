Importar Lista

Como Usar?

Para importar uma lista para o aplicativo, basta você copiar a mensagem ou o texto que o aplicativo gera quando você Compartilha uma de suas listas (veja os botões "Compartilhar" na listagem das listas ou dentro delas). Mensagens criadas manualmente não vão funcionar!

A lista que você deve copiar para o campo abaixo deve se parece mais ou menos com isso aqui: 

      Lista: Padaria
      ===
      - 3× Pão: R$ 6,99 cada;
      - 2× Mortadela;
      - Salame;

      VALOR TOTAL: R$ 20,97
      ===
      Lista gerada pelo app Shopping Cart.

Basta copiar a lista que você recebeu da mesma forma como está no exemplo, colar no campo abaixo e apertar o "Importar"
